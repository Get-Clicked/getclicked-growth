## The Bet

Competitors write the 504 plan. Invincible makes sure it actually happens. We're testing whether this "execution gap" positioning generates demo bookings via Google Ads at a viable CPA.

### The Market Gap

```
                    PLAN DOCUMENT                    PLAN IN ACTION
                    -------------                    --------------
              +----------------------+
              |  Frontline Education |
              |  Embrace Education   |         +---------------------+
              |  Go504 / SpedTrack   |         |                     |
              |                      |  GAP    |    INVINCIBLE       |
              |  "We write the 504"  |<------->|                     |
              |                      |         |  "We execute the    |
              +----------------------+         |   504 -- daily."    |
              |  Magnus Health       |         |                     |
              |  SNAP Health Center  |         |  Logbook / Alerts   |
              |                      |         |  Parent Messaging   |
              |  "We store records"  |         +---------------------+
              +----------------------+
                                          ^
                                          |
                                   Nobody is here.
                                   That's the bet.
```

---

## Hypothesis

**If** we run Google Ads on SPED software and competitor-intercept keywords with execution-gap messaging, **then** district SPED directors and private school admins will book demos at CPA < $200, **because** every buyer already knows their compliance tool doesn't solve daily coordination between nurse, teacher, counselor, and parent.

## Success Criteria

| Condition | Metric | Threshold |

| **WIN** | Cost per demo booking | < $200 over 30 days / $3,000 spend (min 10 demos) |
| **KILL** | CPA | > $400 after $1,500 spend |
| **KILL** | CTR on any ad group | < 1.5% after 2,000 impressions |
